start(utc)

Parameter Type
utc? boolean

Starts this clock's main loop. An event is emitted for every change in the os.date table. The event name is key of the value that changed and the only argument is the corresponding date table.

If utc is passed, UTC time is used; otherwise, local time is used.

Valid events are: sec, min, hour, day, wday, yday, month, year, isdst
Returns: nil

stop()

Stops the clock's main loop immediately.
Returns: nil
