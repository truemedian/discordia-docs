Stopwatch

Used to measure an elapsed period of time. Although nanosecond precision is available, Lua can only reliably provide microsecond accuracy due to the lack of native 64-bit integer support. Generally, milliseconds should be sufficient here.

getTime()

Returns a new Time object that represents the currently elapsed time. This is useful for 'catching' the current time and comparing its many forms as required.
Returns: Time

reset()

Resets the stopwatch. If the stopwatch currently running, the start time is reset to the current time. If the stopwatch is stopped, the start time is reset to the time that the stopwatch was stopped.
Returns: nil

start()

Starts the stopwatch. This function is idempotent, calling it multiple times has no effect.
Returns: nil

stop()

Stops the stopwatch, time elapsed will no longer increase.
Returns: nil
